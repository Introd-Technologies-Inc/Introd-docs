Saved Lists is the older name for what the current Introd product calls [Collections](/workspace/collections). Use Collections for the latest UI, terminology, and workflow guidance.

![Collections in Introd](/images/product/collections.png)

## Common collection types

- investors for a fundraise
- product leaders at target accounts
- alumni network contacts
- strategic partners
- board candidates and advisors

## When to create a collection

Create a collection when a search result should become part of an ongoing workflow. Collections are useful when you need to compare routes, revisit targets, or share context with a team.

## What a good collection contains

- a clear name
- a short purpose
- the people or companies being tracked
- owner and visibility settings
- recent update activity
