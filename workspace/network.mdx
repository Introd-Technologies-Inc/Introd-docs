Network is the operating dashboard for your relationship graph. Use it to understand whether Introd has enough fresh, diverse, and reliable relationship evidence to recommend warm paths.

![Network analytics in Introd](/images/product/network.png)

## What Network answers

- how large your mapped network is
- whether connection growth is improving
- how many introductions are moving through the system
- whether response rates are healthy
- where hidden high-value paths exist
- which relationships are decaying and need re-engagement

## Core metrics

| Metric | Meaning |
| --- | --- |
| Network size | Total mapped relationships across connected sources. |
| New connections | Recently discovered people and edges. |
| Introductions made | Warm introductions completed or advanced during the selected period. |
| Response rate | Share of requests or connector actions that receive a meaningful response. |
| Average time to intro | How long it typically takes to move from request to useful handoff. |
| Network score | Aggregate health score across coverage, freshness, diversity, and activity. |

## Use the right tab

- **Overview** for executive health and trend review.
- **Relationships** for people-level inspection.
- **Opportunities** for reachable targets and hidden paths.
- **Introductions** for request throughput.
- **Path Explorer** for route analysis.
- **Organization Map** for company and community structure.

## Relationship decay

Relationship decay warnings surface contacts that were once useful but have become stale. Treat them as maintenance prompts, not failure states. Re-engage before asking for a high-stakes introduction.

## Operating cadence

Review Network weekly if Introd is part of an active fundraising, hiring, sales, or partnership motion. Review it daily when using Missions or time-sensitive intro requests.
