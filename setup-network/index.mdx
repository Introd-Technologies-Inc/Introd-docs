## Setup checklist

1. Sync LinkedIn.
2. Connect Google or Microsoft evidence.
3. Import contacts.
4. Invite teammates when team coverage matters.
5. Join alumni and investor communities.
6. Create or join groups for ongoing collaboration.

## Read the setup progress

Setup progress is a coverage indicator, not a vanity checklist. A completed step should add usable evidence or access. If a step is checked but search still returns weak paths, review the source status and last sync time.

## What good setup unlocks

- more first-degree coverage
- fresher relationship evidence
- stronger connector recommendations
- better path explanations
- fewer empty or weak-route results
