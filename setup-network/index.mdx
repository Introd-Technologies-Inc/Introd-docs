## Setup checklist

1. Sync LinkedIn.
2. Connect Google or Microsoft evidence.
3. Import contacts.
4. Join alumni and investor communities.
5. Create or join groups for ongoing collaboration.

## What good setup unlocks

- more first-degree coverage
- fresher relationship evidence
- stronger connector recommendations
- better path explanations
- fewer empty or weak-route results
